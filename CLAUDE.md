# 🎯 Site Buffet Regallos - Guia do Projeto

## Skills Disponíveis
- `/simplify` - Simplificar e revisar código
- `/vercel-react-best-practices` - Boas práticas React/Next.js
- `/ui-ux-pro-max` - Design profissional e UX
- `/loop` - Tarefas recorrentes

## Tecnologias
- **Frontend**: Next.js 15 + React 19 + TypeScript
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Funcionalidades Planejadas
- [ ] Homepage com apresentação
- [ ] Cardápio/Pratos
- [ ] Galeria de fotos
- [ ] Contato + WhatsApp

## Estrutura de Pastas
```
src/
├── app/          # Rotas (App Router)
├── components/   # Componentes reutilizáveis
├── styles/       # Estilos globais
└── utils/        # Funções utilitárias
```

## Como Usar as Skills
1. Ao pedir para criar um componente: usa `/ui-ux-pro-max`
2. Ao revisar código: usa `/simplify`
3. Para otimizar React: usa `/vercel-react-best-practices`
